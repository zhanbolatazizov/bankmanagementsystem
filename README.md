# bankmanagementsystem
Bank management system

Technologies:
Java Spring Boot, DTA JPA,
Postgre SQL

Api:
  For bank account
    create bank account: /bankaccount/save/
    get all bank accounts: /bankaccount/
    get bank account by id: /bankaccount/id/
  For transaction
    create transaction: /transaction/save/
    get all transactions: /transaction/
    get transaction by id: /transaction/id/
